# Descrição

**Objetivo**: Criei o blog para treinar meu conhecimento em HTML5/CSS3.

<br>

- Blog responsivo;
- **css grid** e **flexbox**;
- Cores definidas e armazenadas em variáveis no CSS;
- Desenvolvido seguindo a abordagem **mobile first**;
- **media queries**:
    - min-width: 620px;
    - min-width: 960px;
    - min-width: 1200px;
